## Purpose

Optimizing technical documentation for search discoverability.

## When to Apply

Apply to all source documentation and localized versions before completion.

## Steps

1. Define primary and secondary keywords for the topic.
2. Optimize the title and meta description for clarity and search intent.
3. Ensure heading structure (H1, H2, H3) is logical and uses keywords.
4. Add or update internal and external links for relevant context.
5. Verify that structural markers (metadata) are correctly applied.

## Output Format

- Optimized Markdown documentation.

- SEO_REPORT.md detailing changes and keyword performance.
