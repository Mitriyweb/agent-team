## Purpose
Optimizing technical documentation for search discoverability.

## When to Apply
Apply during any documentation creation or update phase to improve reach.

## Steps
1. Define primary and secondary keywords based on search intent.
2. Add or update YAML frontmatter (title, description, keywords).
3. Ensure H1 contains the primary keyword and H2/H3 use variations naturally.
4. Include the primary keyword within the first 100 words of the content.
5. Adapt keywords and metadata for each target locale independently.

## Output Format
- Optimized Markdown documentation files.
- SEO_REPORT.md with summary of changes.
