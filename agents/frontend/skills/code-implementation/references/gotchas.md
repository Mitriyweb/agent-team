# Frontend Project Gotchas

List of project-specific traps and common errors for the frontend team.

- **Environment Variables**: Always read `FRONTEND_FRAMEWORK` and `CSS_APPROACH` before starting development.
- **WCAG 2.1 AA**: All components must adhere to these accessibility standards.
- **Path Spacing**: Be careful with paths that might contain spaces if working across different teams' directories.
- **Linting**: Run `bun run fix` to automatically resolve biome and markdownlint issues.
