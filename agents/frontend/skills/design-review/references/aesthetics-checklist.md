# Premium Design Checklist

This checklist ensures that every UI component or view meets the "WOW" factor and "Premium Design" standards.

## Visual Excellence

- [ ] **System Adherence**: Does it strictly follow the project's discovered design tokens/theme?

- [ ] **Consistent Palette**: Are colors used meaningfully (Primary for actions, etc.)?

- [ ] **Typography**: Is there clear hierarchy? Are specific project-defined headings used?

- [ ] **Gradients**: Are subtle gradients used where appropriate to add depth?

- [ ] **Soft UI**: Are border-radii appropriately soft (e.g., 12px)?

- [ ] **Glassmorphism**: Is backdrop-blur used for overlays or headers?

## Dynamic Interactions

- [ ] **Snappy Transitions**: Are all state changes (e.g., visibility) animated (150ms-300ms)?

- [ ] **Micro-animations**: Do buttons scale on active? Do cards lift on hover?

- [ ] **Loading states**: Are skeleton screens or sleek loaders used?

- [ ] **Empty states**: Are they visually engaging (not just text)?

## Polish

- [ ] **Consistent Spacing**: Is the spacing unit (8px) followed strictly?

- [ ] **Shadows**: Are shadows soft and multi-layered, not harsh blobs?

- [ ] **Accessibility**: Does it still meet WCAG 2.1 AA after applying aesthetic effects?
