Autonomous development team powered by Claude Code. Drop this into any project and let the agents work through your roadmap.

Why Agent Team?

Autonomous

Agents work through your roadmap automatically, from architecture to testing.

Framework Aware

Human-in-the-loop

Keeps you in control with interactive approvals for critical steps.

Installation

macOS only (Apple Silicon & Intel). Linux and Windows support is planned.

Option 1: One-line install (Recommended)

Detects your platform, downloads the binary, and installs to ~/.local/bin/.

curl -fsSL https://raw.githubusercontent.com/Mitriyweb/agent-team/main/install.sh | bash

Option 2: npx (no install)

npx @mitriyweb/agent-team init

Option 3: Clone from source

git clone https://github.com/Mitriyweb/agent-team.git
cd agent-team && bun install && bun run build
ln -sf $(pwd)/dist/agent-team ~/.local/bin/agent-team

Quickstart

1. Initialize your project:

agent-team init

2. Create a plan (interactive — select or create OpenSpec change):

agent-team plan

3. Execute tasks (interactive — select which change to run):

agent-team run --all

Planning Modes

Built-in

Write a ROADMAP.md, run agent-team plan. Tasks decomposed into tasks/plan.md with priorities and dependencies.

OpenSpec

Structured spec-driven planning. Each change gets a proposal, design, and task checklist. Agents execute directly from openspec/changes/NAME/tasks.md.

Ready-to-use Teams

Software Development

Team Lead, Architect, Developer, Reviewer, QA. Full lifecycle from design to testing.

Frontend

UI Architect, Frontend Dev, Design Reviewer, QA. Focused on components, accessibility, and visual quality.

Localization

Technical Writer, Localizer, SEO Specialist, QA. Content creation and translation pipeline.

Use Cases & Examples

Reliable Features

Software Development: Uses the Proof-Loop methodology. Architect freezes SPEC.md, Developer provides EVIDENCE.md, and QA issues a VERDICT.json after fresh verification.

Accessible UI

Frontend: Team ensures WCAG 2.1 AA compliance. UI Architect defines tokens, Dev builds components, and AQA runs visual regression and performance audits.

Multi-language Support

Localization: Synchronized workflow where Technical Writer, Localizer, and SEO Specialist work in parallel to keep global documentation up-to-date.

Monitoring

Every tool call is logged to .claude-loop/audit/audit.jsonl with role, agent name, tool, and phase. Task logs and cost reports are saved to .claude-loop/logs/ and reports/.

agent-team audit    # Show audit report
